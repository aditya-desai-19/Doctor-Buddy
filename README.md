# **DoctorBuddy**

DoctorBuddy is a modern **Patient Management System** designed to simplify and streamline clinic operations for healthcare professionals. It helps manage patients, treatments, and payments efficiently, with a focus on ease-of-use and security.

---

## **Features**

- 🏥 **Patient Management**: Add, edit, and view patient profiles.
- 💊 **Treatment Tracking**: Create and manage treatment records for each patient.
- 💳 **Payment Handling**: Record payments, handle partial and full transactions, and view payment history.
- 🗂️ **Soft Deletion**: Records are soft-deleted for safety, allowing recovery when needed.

---

## **Tech Stack**

| Technology | Description |
|:-----------|:------------|
| **Backend** | Node.js + Express |
| **Database** | PostgreSQL (via Prisma ORM) |
| **Frontend** | Next.js |
